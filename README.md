# Swigato User Guide Portfolio Project

This repository contains a comprehensive, task-based user guide for a fictional food delivery application, **Swigato**. 

I built this project to demonstrate my proficiency in modern technical writing workflows and "Docs-as-Code" methodologies. 

## 🛠️ Tools & Technologies Used
* **Authoring:** Markdown
* **Version Control:** Git & GitHub
* **Environment:** Visual Studio Code (VS Code)
* **Architecture:** Task-based information architecture separated by user journey phases.

## 📁 Project Structure
* `README.md`: Project overview and technical details.
* `docs/`: Contains the actual end-user documentation.
  * `1-Getting-Started/`: Account creation and onboarding.
  * `2-Ordering-Food/`: Core application workflows.
  * `3-Managing-Orders/`: Post-purchase troubleshooting.
